# ⚡ JobAgent AI — Automated Job Application System

An AI-powered job discovery, matching, and auto-apply engine built for finance professionals.

## 🚀 What It Does
- 🔍 Scrapes jobs from multiple platforms (LinkedIn, Naukri, APIs)
- 🧠 Matches jobs with your resume using NLP (Sentence Transformers)
- 🎯 Scores jobs based on relevance
- 🤖 Automatically applies to selected roles
- ✍️ Generates AI-based cover letters

## 🧠 Tech Stack
- Python
- Selenium (Automation)
- Sentence Transformers (AI Matching)
- RapidAPI (Job Data)
- Groq API (LLM)

## 🔥 Key Features
- Resume-based semantic matching
- AI scoring engine with custom boosting
- Multi-platform job discovery
- Automated job application bot
- Cover letter generation

## ⚠️ Current Limitations
- Auto-apply may fail on dynamic websites (in progress)
- Requires login session for some platforms

## 🚀 Future Improvements
- Smart selector engine for auto-apply
- Retry logic for failed applications
- AI-based UI detection
- Chrome extension integration
